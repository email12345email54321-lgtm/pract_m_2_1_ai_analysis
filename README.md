## Цель работы
Провести сравнительный анализ функциональности зарубежного и отечественного чат-бота с искусственным интеллектом. Смоделировать сценарии использования для горнодобывающего или нефтегазового сектора и оценить эффективность работы каждого из ботов. Результаты опубликовать в открытом репозитории GitHub.

## Выбранные чат-боты
- **Зарубежный:** ChatGPT (OpenAI, веб-версия, модель GPT-4o / o1 на февраль 2026)
- **Отечественный:** GigaChat (Сбер, веб-версия, модель Pro/Max)

## Выбранные кейсы
1. **Диагностика насоса НШ-32У**  
   Падение давления, перегрев корпуса, повышенная вибрация. Пошаговый алгоритм диагностики с опорой на физику процессов.

2. **Задымление в цехе дробления обогатительной фабрики**  
   Чек-лист первоочередных действий рядового сотрудника + текст объявления по громкой связи.
