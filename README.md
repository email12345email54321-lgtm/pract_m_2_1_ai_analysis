Исследование возможностей чат-ботов с искусственным интеллектом  
для решения задач добывающей промышленности  
(на примере сравнительного анализа кейсов)

## Цель работы
Провести сравнительный анализ функциональности зарубежного и отечественного чат-бота с искусственным интеллектом. Смоделировать сценарии использования для горнодобывающего или нефтегазового сектора и оценить эффективность работы каждого из ботов. Результаты опубликовать в открытом репозитории GitHub.

## Выбранные чат-боты
- **Зарубежный:** ChatGPT (OpenAI, веб-версия, модель GPT-4o / o1 на февраль 2026)  
- **Отечественный:** GigaChat (Сбер, веб-версия, модель Pro/Max)

## Выбранные кейсы
1. Диагностика насоса НШ-32У (падение давления, перегрев, повышенная вибрация).  
2. Задымление в цехе дробления обогатительной фабрики (чек-лист действий сотрудника + текст оповещения по громкой связи).

## Ход работы (по методичке)
1. Выбраны представители зарубежных (ChatGPT) и отечественных (GigaChat) LLM-чат-ботов.  
2. Выявлены ключевые проблемные области: диагностика оборудования и производственная безопасность.  
3. Разработаны одинаковые запросы для решения задач в рамках кейсов.  
4. Проведено тестирование, зафиксированы результаты работы обоих ботов (в папке results/).  
5. Подготовлен отчёт с анализом и выводами.

## Методические указания (что выполнено)
- Использованы минимум два сервиса: зарубежный и отечественный.  
- Кейсы выбраны из предложенных направлений (ремонт оборудования и производственная безопасность).  
- Запросы отправлены в новых чатах без истории.  
- Результаты сохранены в виде PDF и скриншотов.

## Структура репозитория
- `cases/` — описание кейсов и запросов  
- `prompts/` — тексты запросов  
- `results/` — ответы ботов (PDF, скриншоты)  
- `analysis/` — сравнительный анализ и выводы  
- `README.md` — этот документ
